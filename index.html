Qué es CSS GRID y por qué mola

CSS GRID o GRID es el primer sistema dedicado a resolver estructuras complejas de rejillas o grids. Bueno, complijas y no tan complejas ;)

Qué necesitamos

HTML

<div class="container">
	<div class="item">1</div>
	<div class="item">2</div>
	<div class="item">3</div>
</div>

CSS

.container {
	display: grid | inline-grid | subgrid;
}
Estilos del contenedor GRID

grid-template-···

Establece la rejilla

grid-template-rows: 150px 1fr 100px;
grid-template-columns: 1fr 200px;

grid-template-rows: repeat(4, 100px);
grid-template-columns: minmax(auto, 50%) 1fr 3em;

grid-template-areas: "header header" "content sidebar" "footer footer";

Base CSS GRID en Codepen, yay!

Estilos del contenedor GRID

grid-···-gap

Define el espaciado entre columnas y filas.

grid-row-gap: 20px;
grid-column-gap: 5rem;

grid-gap: 5% 10%;
grid-gap: 20px;
Estilos del contenedor GRID

grid-auto-···

Establece los valores por defecto para columnas o filas en caso de que se necesiten más de las definidas.

Además puedes marcar la dirección principal del elemento GRID

grid-auto-rows: 100px;
grid-auto-columns: 110px;
grid-auto-flow: row | columns
Estilos del item GRID

Posicionando elementos

Marca cómo se va a posicionar un elemento en nuestra rejilla.

grid-column-start: 2;
grid-column-end: 3;

grid-row-start: 1;
grid-row-end: 4;

grid-row: 1 / 4;
grid-column: 2 / 3;

grid-row: 2 / span 3;
grid-column: span 3;

grid-area: header
